Olty Ürün Test Checklist

Bu liste, ürünü uçtan uca test etmek ve MVP kalitesini gerçek kullanım gibi değerlendirmek için hazırlandı.

Kapsam: Auth, onboarding, catch log, feed, yorum, beğeni, profil, mesajlar, bildirimler, günlük oyunlar, admin, güvenlik, cihaz testleri.
Önerilen test hesapları: `user_a`, `user_b`, `user_c`, `admin`
Önerilen cihazlar: en az 1 iPhone, 1 Android

1. Test Hazırlığı

  • 4 test hesap hazır: public kullanıcı, private kullanıcı, yeni kullanıcı, admin
  • En az 2 gerçek cihaz hazır: iPhone ve Android
  • Yavaş internet ve internet kes / geri gel senaryosu hazırlanmış
  • En az 15 catch içeren test veri seti hazırlanmış
  • Fotoğraflı ve fotosuz catch örnekleri hazır
  • Public ve private catch karışık veri hazır
  • En az 5 gear item ve birkaç mesaj / yorum / beğeni var
  • Push izni açık ve kapalı iki ayrı cihaz senaryosu hazır
  • Konum izni açık ve kapalı iki ayrı cihaz senaryosu hazır

2. Smoke Test

  • Uygulama cold start'ta crash olmadan açılıyor
  • Login olmayan kullanıcı auth ekranına yönleniyor
  • Login olan kullanıcı tab yapısına yönleniyor
  • Feed, explore, log, map, notifications ve profile ekranları açılıyor
  • Blank screen veya navigation dead-end yok
  • Session kapatıp açınca kullanıcı oturumu doğru restore oluyor

3. Auth ve Onboarding

  • E-posta + şifre kayıt çalışıyor
  • Hatalı e-posta validasyonu çalışıyor
  • Şifre validasyonu doğru hata veriyor
  • Duplicate username engelleniyor
  • Username kuralı çalışıyor: 3-30 karakter, yalnızca a-z, 0-9, _
  • KVKK onayı olmadan onboarding tamamlanamıyor
  • Şehir seçimi kaydediliyor
  • Balıkçılık tipi multi-select doğru kaydediliyor
  • Onboarding skip akışı bozulmadan tamamlanıyor
  • Forgot password ve logout akışları doğru çalışıyor

4. Feed ve Explore

  • Takip edilen kullanıcıların catch'leri feed'de görünüyor
  • Kendi catch'leri feed'de görünüyor
  • Yeni kullanıcıda keşfet feed'i açılıyor
  • Sıralama kronolojik ve pagination düzgün
  • Refresh sonrası duplicate kart oluşmuyor
  • Blur placeholder'dan gerçek görsele geçiş düzgün
  • Private hesap catch'leri yetkisiz kullanıcıya görünmüyor
  • show_exact_location false olan catch tam konum göstermiyor

5. Catch Oluşturma ve Düzenleme

  • Fotosuz catch oluşturulabiliyor
  • Fotoğraflı catch oluşturulabiliyor
  • JPEG ve PNG yükleniyor
  • Büyük dosya senaryosu düzgün ele alınıyor
  • Upload fail olursa catch foto'suz kaydedilebiliyor
  • Tür seçerek catch eklenebiliyor
  • Tür seçmeden custom tür ile catch eklenebiliyor
  • Boy, ağırlık ve not alanı validasyonları doğru
  • Public/private toggle ve catch release toggle doğru çalışıyor
  • Konumu cihazdan çekme ve manuel konum girişi çalışıyor
  • Catch edit ve delete akışı çalışıyor
  • Başkasının catch'ini edit / delete yapamıyorsun

6. Catch Detay, Beğeni ve Yorum

  • Catch detay ekranı açılıyor
  • Public catch herkesçe görüntülenebiliyor
  • Private catch sadece yetkililerce açılıyor
  • Like optimistic update doğru
  • Unlike doğru
  • Double-tap like çalışıyor
  • Duplicate like oluşmuyor
  • Yorum ekleme çalışıyor
  • 280 karakter üzeri yorum engelleniyor
  • Kendi yorumunu silebiliyorsun
  • Kendi catch'indeki başkasının yorumunu silebiliyorsun
  • Yetkisiz kullanıcı yorum silemiyor

7. Profil ve Sosyal

  • Kendi profil ekranı açılıyor
  • Başka kullanıcı profili açılıyor
  • Takip et / takipten çık çalışıyor
  • Private profil takip mantığı doğru
  • Follower / following listeleri doğru
  • Profil edit, avatar güncelleme ve bio limiti doğru çalışıyor
  • Rozetler ve catch grid düzgün görünüyor

8. Bildirimler ve Mesajlaşma

  • In-app notifications listeleniyor
  • Like, comment, follow, badge ve level up bildirimleri doğru düşüyor
  • Bildirim okundu / okunmadı durumu doğru
  • Notification settings ekranı çalışıyor
  • Conversation list açılıyor
  • Yeni mesaj gönderilebiliyor
  • Karşı taraf mesajı görüyor
  • Paylaşım üzerinden catch gönderme akışı çalışıyor
  • Yetkisiz kullanıcı başka conversation'a mesaj yazamıyor

9. Günlük Oyunlar

  • Günün sorusu görünüyor
  • Günün fish challenge'ı görünüyor
  • Aynı kullanıcı aynı soruya ikinci kez cevap veremiyor
  • Cevap verince UI durumu doğru güncelleniyor
  • Reveal sonrası doğru / yanlış hesap doğru
  • XP ödülü doğru işleniyor
  • 3 gün üst üste cron ile kayıt oluşuyor
  • Bugünkü kayıtlar Supabase'te var
  • Reveal cron'u doğru saatte çalışıyor

10. Leaderboard, Harita, Hava

  • Leaderboard açılıyor
  • Weekly catch count doğru
  • Biggest fish sıralaması doğru
  • All-time XP doğru
  • City / country filtreleri doğru
  • Map ekranı açılıyor
  • Yaklaşık konum masking doğru
  • Hava widget'ı ve balık skoru düzgün çalışıyor

11. Gear, Fishdex, Search

  • Gear list açılıyor
  • Yeni gear ekleniyor
  • Gear edit ve delete çalışıyor
  • Tier seçimi kaydediliyor
  • Fishdex ekranları açılıyor
  • Arama ekranı kullanıcı buluyor
  • Kendi ve başka kullanıcının fishdex görünümü doğru

12. Admin

  • Admin dışı kullanıcı admin route'a giremiyor
  • Admin panel açılıyor
  • Daily question admin ekranı çalışıyor
  • Fish challenge admin ekranı çalışıyor
  • Weekly challenge admin ekranı çalışıyor
  • Admin işlem sonrası public UI doğru güncelleniyor

13. Güvenlik ve Yetki

  • Private catch API'den yetkisiz okunamıyor
  • Başkasının storage alanına yazılamıyor
  • Kendi olmayan kaydı update / delete yapılamıyor
  • Logout sonrası session temizleniyor
  • identify-fish auth, quota ve size limitleri çalışıyor
  • Mesaj, yorum, beğeni ve bildirim RLS kuralları doğru

14. Dayanıklılık ve Gerçek Cihaz Testi

  • İnternet kesikken kontrollü hata veriyor
  • Retry sonrası veri toparlıyor
  • Çok hızlı tıklamada duplicate kayıt oluşmuyor
  • Arka plana alıp dönünce state bozulmuyor
  • Uzun listelerde performans bozulmuyor
  • Görsel olmayan catch kartları düzgün render oluyor
  • iPhone login / logout
  • iPhone catch upload / push / location
  • Android login / logout
  • Android catch upload / push / location
  • KVKK consent akışı uçtan uca doğru
  • Account deletion akışı uçtan uca doğru

15. Çıkış Kriteri

  • P0 crash yok
  • P1 bloklayıcı bug yok
  • Günlük oyun 3 gün üst üste otomatik oluşuyor
  • Tüm core akışlar gerçek cihazda geçti
  • Yetki / RLS ihlali bulunmadı
  • Veri kaybı veya duplicate kayıt sorunu yok

Dosya kaynağı: Olty repo feature ve launch checklist dokümanları üzerinden derlenmiştir.
